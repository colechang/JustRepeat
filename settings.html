Extension Settings

Icons and Their Meanings

  • Play Icon: Click this icon to play the loop.
  • Delete Icon: Click this icon to delete the loop.
  • Label Icon: Click this icon to add a label to the loop.

How to Use the Extension

  1. Open a YouTube video page in your browser.
  2. Use the YouTube video controls to seek to the desired loop start position.
  3. Click the "Loop" button to set the loop start position.
  4. To add a label to the loop:
    1. Click the "Label Icon" for the specific loop.
    2. Enter the desired label in the input field.
    3. Click "Save" to save the label.
  5. Use the YouTube video controls to seek to the desired loop end position.
  6. Click the "Loop" button again to set the loop end position.
  7. The loop will now play continuously between the start and end positions.
  8. To play, delete, or modify a loop:
    1. Use the respective icons for the loop you want to interact with.
